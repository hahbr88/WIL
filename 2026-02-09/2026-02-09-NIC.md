# 2026-02-09-NIC

## 🌐 네트워크 인터페이스 카드 (NIC)

### 1️⃣ NIC란?

- 컴퓨터를 네트워크에 연결해주는 하드웨어 장치
- **Network Interface Card**
- 네트워크 통신의 최소 단위
- 유·무선 통신 모두 포함

👉 “컴퓨터가 네트워크 세계로 나가는 문”

### 2️⃣ NIC의 핵심 역할

1. 물리적 연결
    - 유선: 랜 케이블(RJ45)
    - 무선: Wi-Fi, Bluetooth

2. 데이터 프레임화
    - 상위 계층 데이터 → 프레임으로 캡슐화
    - 반대로 수신 시 프레임 해제

3. MAC 주소 관리
    - 고유한 MAC 주소 보유
    - 같은 네트워크에서 장치 식별

4. 전기 신호 ↔ 디지털 데이터 변환
    - 비트 스트림을 신호로 변환
    - 수신 시 다시 데이터로 복원

### 3️⃣ NIC가 관여하는 OSI 계층

```bash
7  응용
6  표현
5  세션
4  전송 (TCP/UDP)
3  네트워크 (IP)
2  데이터링크  ← NIC 핵심
1  물리        ← NIC 핵심
```

👉 2계층 + 1계층 담당

### 4️⃣ NIC 내부 구성 (개념)

- MAC 컨트롤러
- PHY 칩 (물리 신호 처리)
- 버퍼(송·수신 큐)
- EEPROM (MAC 주소 저장)

### 5️⃣ NIC 동작 흐름 (송신 기준)

- OS가 데이터 전달
- NIC가 프레임 생성
- 출발 MAC / 목적 MAC 설정
- CRC 추가
- 전기/무선 신호로 전송
- 수신은 이 과정의 역순

### 6️⃣ NIC 종류

- 연결 방식
  - 유선 NIC (Ethernet)
  - 무선 NIC (Wi-Fi)
- 형태
  - 메인보드 내장
- PCIe 확장 카드
- USB-to-LAN 어댑터

### 7️⃣ 속도 & 규격

- 10 / 100 / 1000 Mbps
- 2.5G / 10G Ethernet
- Full-Duplex 지원

### 8️⃣ NIC와 IP 주소 관계

- MAC 주소: NIC에 고정
- IP 주소: 소프트웨어 설정 (변경 가능)

👉 NIC는 MAC, OS는 IP

### 9️⃣ 실제 환경에서 중요한 포인트

```bash
서버: 멀티 NIC
가상화: 가상 NIC(vNIC)
보안: MAC 스푸핑 가능
성능: 오프로드 기능(TSO, LRO)
```
