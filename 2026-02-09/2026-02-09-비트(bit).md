# 2026-02-09-비트(bit)

## 1️⃣ 왜 저장소 단위는 2의 제곱인가?
컴퓨터 비트(bit, Binary Digit)는 0 또는 1의 값을 가지는 정보의 최소 단위로, 디지털 데이터를 저장하고 처리하는 가장 기본적인 요소, 비트는 0 아니면 1 두 가지 상태뿐이다

그래서 컴퓨터는 자연스럽게:
> 2¹ = 2
> 
> 2² = 4
> 
> 2¹⁰ = 1024
> 
> 2²⁰ = 1,048,576
> 
> 2³⁰ = 1,073,741,824

이렇게 2의 제곱 단위로 데이터를 다루는 게 가장 효율적.

👉 그래서 전통적으로

> 1KB = 1024B
> 
> 1MB = 1024KB
> 
> 1GB = 1024MB

이 방식이 컴퓨터 내부(메모리, 파일시스템) 기준.


## 2️⃣ 그런데 왜 1TB 저장장치를 열어보면 931GB 일까?

저장장치 제조사(SSD, HDD) 기준
제조사는 사람이 이해하기 쉬운 10진수를 씀.

> 1KB = 1000B
> 
> 1MB = 1000KB
> 
> 1GB = 1000MB
> 
> 1TB = 1000GB = 1,000,000,000,000B

👉 이게 마케팅 + 국제 단위계(SI) 기준.


## 3️⃣ 그래서 1TB가 931GB로 보이는 이유

>📦 디스크에 실제로 들어 있는 용량
>
> 1TB = 1,000,000,000,000 bytes

> 🧠 컴퓨터가 쓰는 단위 (2의 제곱)
> 
> 1GiB = 1,073,741,824 bytes

> 🔢 변환
> 
> 1,000,000,000,000 ÷ 1,073,741,824 ≈ 931.32

👉 그래서 운영체제에서는 약 931GB로 보이는 것.


## 4️⃣ 사실 정확히 말하면 “GB”가 아니라 “GiB”

```bash
단위 기준 바이트
GB	10진 1,000,000,000
GiB	2진 1,073,741,824

근데 대부분 OS가 표시는 GB라고 해놓고 실제 계산은 GiB로 한다
(macOS는 최근엔 표기까지 10진으로 맞춰줌, Windows는 아직 혼용)
```

## 5️⃣ 세 줄 요약
저장장치는 10진수(1000) 기준으로 용량을 표시하지만,
컴퓨터는 2진수(1024) 기준으로 계산하기 때문에
1TB 저장소가 약 931GB로 보인다.
