# 2026-02-09-컴퓨터구조

## 컴퓨터 구조

컴퓨터의 기본 구조는 크게 보면 “입력 → 처리 → 저장 → 출력” 흐름으로 이해하자

### 1️⃣ 입력 장치 (Input Device)

- 사용자가 데이터나 명령을 컴퓨터에 넣는 역할
  - 키보드, 마우스, 터치스크린, 스캐너, 마이크  
  👉 “컴퓨터야, 이거 해줘”라고 알려주는 창구

### 2️⃣ 중앙처리장치(CPU, Central Processing Unit)

컴퓨터의 두뇌, 입력받은 명령을 해석하고 계산함

- CPU 내부 구성
  - 제어장치(CU) → 명령을 해석하고 각 장치에 지시
  - 연산장치(ALU) → 덧셈, 뺄셈, 비교 같은 계산 수행
  - 레지스터(Register) → 아주 빠른 임시 저장 공간

  👉 “생각하고 판단하는 곳”

### 3️⃣ 기억 장치 (Memory / Storage)

데이터를 저장하는 역할

(1) 주기억장치

- RAM : 전원이 꺼지면 내용이 사라짐, 실행 중인 프로그램과 데이터 저장
- ROM : 전원이 꺼져도 유지, 부팅 관련 기본 정보 저장

(2) 보조기억장치

- SSD / HDD
- USB
- 외장하드

👉 “RAM = 책상 위 메모”  
👉 “SSD = 서랍 속 파일”

### 4️⃣ 출력 장치 (Output Device)

처리 결과를 사람이 볼 수 있게 출력

모니터, 프린터, 스피커 등  
👉 “결과 보여주는 창구”
